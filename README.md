# MLCoreBasics
Basics of Python data handling, visualization, and ML fundamentals

## Structure
- `notebooks/` → Jupyter notebooks for daily/weekly exercises
- `src/` → Reusable Python modules (data loading, plotting, training, evaluation)
- `notes/` → Weekly Markdown notes summarizing learnings
- `datasets/` → Example datasets

## Goals
- Refresh Python (NumPy, Pandas, Matplotlib/Seaborn)
- Learn supervised ML models (linear regression, logistic regression, decision trees)
- Understand evaluation metrics (accuracy, precision, recall, F1)
- Build a mini-project with clear documentation

## Resources
- Andrew Ng’s Machine Learning Specialization
- *Hands-On Machine Learning with Scikit-Learn & TensorFlow* by Aurélien Géron
- Kaggle datasets for practice
